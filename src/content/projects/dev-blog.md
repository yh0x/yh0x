## 개발자 블로그

Astro 프레임워크와 Nano 테마를 기반으로 만든 개인 블로그입니다.

### 주요 기능

- **Blog**: 개발, AI, 툴 관련 포스팅
- **Projects**: 사이드 프로젝트 소개
- **Work**: 경력 사항 정리

### 기술 스택

- **Framework**: Astro 5.x
- **Styling**: TailwindCSS
- **Font**: Pretendard (본문), JetBrains Mono (코드)
- **Deployment**: GitHub Pages

### 개발 후기

정적 사이트 생성기인 Astro를 사용해서 빠르고 가벼운 블로그를 구축했습니다. Obsidian과 연동하여 마크다운으로 편리하게 포스팅을 관리할 수 있습니다.
