### 주요 업무

- React Native를 활용한 크로스플랫폼 모바일 앱 개발
- Flutter 기반 UI 컴포넌트 개발
- 물류 관리 시스템 프론트엔드 구현
- 성능 최적화 및 코드 품질 개선
