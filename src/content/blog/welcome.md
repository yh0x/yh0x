## 블로그를 시작하며

안녕하세요! 모바일 앱 개발자 윤여현입니다.

이 블로그에서는 개발하면서 배운 것들, 사용하는 도구들, 그리고 AI 관련 정보들을 공유할 예정입니다.

### 다룰 주제들

- **개발 정보**: React Native, Flutter, 웹 개발 관련 팁과 트릭
- **AI 도구**: 개발에 활용하는 AI 도구들 소개
- **사이드 프로젝트**: 개인 프로젝트 개발 후기와 기록

앞으로 꾸준히 글을 올려보겠습니다. 🚀

```typescript
// 코드 블록 예시
const greeting = (name: string): string => {
  return `안녕하세요, ${name}!`;
};

console.log(greeting("World"));
```
